# Flutter Start Prompt

Use this for Flutter and Dart implementation work.

These prompts assume the global install has already been applied.

## Prompt

```text
$godmode-workflow
$flutter-dart

GODMODE FLUTTER

Goal: <goal>

Inspect the current workspace first.
Loop: research -> plan -> build -> validate.
```

## Best for

- Flutter UI work
- Dart package changes
- repository and state-flow refactors
- cross-platform app work that spans mobile, desktop, or web

## Optional extras

- affected package, app, or directory
- required validation command such as `flutter analyze`, `flutter test`, or `dart analyze`
- platform scope such as `iOS`, `Android`, `macOS`, or `web`
